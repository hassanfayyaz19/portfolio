# Hassan Fayyaz - Portfolio

A modern, responsive portfolio website built with Vue 3, Vite, and Tailwind CSS.

## Tech Stack

- **Vue 3** - Composition API
- **Vite** - Build tool
- **Tailwind CSS** - Styling

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Project Structure

```
src/
├── components/     # Vue components
├── data/          # Portfolio content (projects, experience, etc.)
├── App.vue
├── main.js
└── style.css
```

## Customization

Edit `src/data/portfolio.js` to update:
- Social links (LinkedIn, GitHub, email)
- Hero section (name, title, tagline)
- About section (bio, skills)
- Projects
- Work experience
